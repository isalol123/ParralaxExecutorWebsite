Common Issues

Before reaching out, check these common solutions:

Injection Failed

- Run Parallax as administrator - Make sure Roblox is fully loaded before injecting - Disable antivirus temporarily during injection - Restart both Roblox and Parallax

Scripts Not Working

- Verify the script is compatible with current Roblox version - Check if the script requires specific game conditions - Try running the script in a private server first - Make sure you're using the latest version of Parallax

Performance Issues

- Close unnecessary programs while using Parallax - Lower Roblox graphics settings - Ensure your PC meets minimum requirements - Update your graphics drivers

Antivirus Detection

- Add Parallax folder to antivirus exclusions - Temporarily disable real-time protection during download - Use Windows Defender instead of third-party antivirus - Download from official sources only

Update Problems

- Download the latest version manually - Clear temporary files and restart - Check Discord for known update issues - Contact support if problems persist
